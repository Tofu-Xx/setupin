## 😏 为什么选择 setupin

**setupin** 允许你在 HTML 中编写 Vue 的 [\<script setup\>](https://vuejs.org/api/sfc-script-setup.html)。

- **便于学习**：为初学者提供一个友好的环境，帮助他们轻松上手 Vue 的核心用法。
- **简易开发**：提供便捷的方式，助力快速开发小网页，无需繁琐的配置。
- **快速体验**：让用户可以快速在 HTML 中尝试 Vue 的新特性，感受其魅力。
